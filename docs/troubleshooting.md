# Troubleshooting

Common issues and solutions.

| Problem | Possible Cause | Solution |
|----------|----------------|-----------|
| No video feed | Camera not detected | Check ribbon connection, reboot Pi |
| GUI fails to connect | Wrong IP/hostname | Verify `ping my-project-pi.local` |
| Audio noise | Electrical interference | Add software filter or reposition mic |
| AI misclassifications | Poor dataset | Retrain or adjust confidence threshold |
